• fyb-apps - модули приложений
    • dummy - типовой шаблон модуля приложения
      • config - конфигурационные файлы модуля приложения
      • templates - HTML-шаблоны модуля приложения, если существуют, то перезаписывают базовые шаблоны
          • container.sidebar.html - шаблон боковой панели модуля приложения
        • container.main.html - шаблон основного содержимого модуля приложения
      • index.php - диспетчер запросов модуля приложения
  • fyb-config - системные конфигурационные файлы
    • db.config.php - параметры подключения к базе данных
    • variables.php - различные переменные
  • fyb-content - css-стили, bootstrap, шаблоны, изображения, содержимое, подключаемые на страницах разных приложений
    • config - конфигурационные файлы приложения
    • templates - базовые HTML-шаблоны приложения
      • main.head.html - базовый шаблон шапки сайта
      • main.navbar.html - базовый шаблон верхней панели
      • main.container.sidebar.html - базовый шаблон боковой панели
      • main.container.main.html - базовый шаблон основного содержимого
      • main.footer.html - базовый шаблон подвала сайта
  • fyb-data - файлы, необходимые для работы приложений
  • fyb-system - файлы с основным PHP кодом, подключаемым на страницах разных приложений
    • db.connect.php - определяет основное поведение подключений к базе данных
    • functions.php - основные функции приложения
  • index.php - основной диспетчер запросов приложения
